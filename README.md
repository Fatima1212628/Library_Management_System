# Library Management System

A robust library management system built with **Django 6.0**, featuring automated book issue/return mechanics, active copy tracking.

## 🚀 Features
*   **Book Issue & Return:** Clean dashboard interface to track which member has borrowed which book.
*   **Inventory Automation:** Automatically decrements `available_copies` on issue and increments it back upon return.
*   **Polished Django Admin:** Customized list views with green/red status indicators for fine tracking.

## 🛠️ Tech Stack
*   **Backend:** Python 3.13+, Django 6.0+
*   **Database:** SQLite (Local development)
